
This file contains a list of cmake variables (and their description)
that can be turned on/off by the user to enable/disable certain features
at compile time since these are pre-processor directives.


- HAVE_TRILINOS
default = ON if TPL_ENABLE_TRILINOS = ON, OFF otherwise
description: self-explanatory

- HAVE_ANASAZI_TSQR
default = OFF
can be turned on with: -D HAVE_ANASAZI_TSQR:BOOL=ON
description: enables the use of the Anasazi TSQR classes
prerequisites: HAVE_TRILINOS must be on

- HAVE_BLAZE
default = ON if TPL_ENABLE_BLAZE = ON, OFF otherwise
description: self-explanatory

- HAVE_MPI
default = ON if TPL_ENABLE_MPI = ON, OFF otherwise
description: self-explanatory

- HAVE_ARMADILLO
default = ON if TPL_ENABLE_ARMADILLO = ON, OFF otherwise
description: self-explanatory

- HAVE_CPP14
default = ON if ${CMAKE_CXX_STANDARD} EQUAL "14", OFF otherwise
description: if compiler supports C++14 standard

- DEBUG_PRINT
default = OFF
description: to turn on/off print statemetns for debugging
