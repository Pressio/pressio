
``Time``
========

.. code-block:: cpp

    template <class T>
    concept Time =
      std::regular<T> &&
      std::totally_ordered<T> &&
      requires(T t1, T t2)
      {
        {t1 + t2}  -> std::same_as<T>;
        {t1 += t2} -> std::same_as<T &>;
      }
      // axioms
      // {t2 == t1 + dt} => ;
      // t1==t2 => represent the same time instance
