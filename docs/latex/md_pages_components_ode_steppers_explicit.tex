Applicable to systems of the form\+: \[ \frac{d \boldsymbol{y}}{dt} = \boldsymbol{f}(\boldsymbol{y},t; ...) \]

where $y$ is the state, $f$ is the RHS (also called velocity below), $t$ is time.~\newline
 Explicit methods calculate the state of a system at a later time from the state of the system at the current time and potentially previous times.\hypertarget{md_pages_components_ode_steppers_explicit_autotoc_md34}{}\doxysection{API\+:}\label{md_pages_components_ode_steppers_explicit_autotoc_md34}
