When you create a problem object, pressio behind the scenes checks types, and instantiates everything that is needed to define a problem. Practically speaking, at the lowest-\/level, a Galerkin problem reduces to creating a \char`\"{}custom\char`\"{} stepper to advance in time. This is exactly how pressio implements this\+: when you create a Galerkin problem, pressio behind the scenes creates this custom stepper object that you can use. You don\textquotesingle{}t know to know or rely on the details of the stepper or how this is done, because these are problem-\/dependent and we reserve the right to change this in the future.

All you need to know to use a problem is that it has the following interface\+:
