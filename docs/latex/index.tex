This structure benefits\+:
\begin{DoxyItemize}
\item Maintability/\+Modularity\+: the components depend on one another through well-\/defined public interfaces, and appropriate namespaces are used to properly scope the code.
\item Selective usability\+: users can leverage invidual functionalities. This allows finer-\/grained control on what you include and use.
