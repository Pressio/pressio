Public namespace\+: {\ttfamily pressio\+::ode}\hypertarget{md_pages_components_ode_autotoc_md28}{}\doxysection{Overview}\label{md_pages_components_ode_autotoc_md28}
The ode package provides classes for doing time integration. Specifically, the design of the package is rooted on the following main ideas\+:
\begin{DoxyItemize}
\item we separate the concept of a \char`\"{}stepper\char`\"{} from the actual \char`\"{}advancer\char`\"{}\+: a stepper class is responsible for knowing how to perform a step in time, while the \char`\"{}advancer\char`\"{} (or integrator) is responbile of knowing how many times the stepper should be called.
\item we use a policy-\/based design for evaluating the required operators, such that this allows great flexiblity on using this time integration package.
\end{DoxyItemize}\hypertarget{md_pages_components_ode_autotoc_md29}{}\doxysection{Explicit Methods}\label{md_pages_components_ode_autotoc_md29}
The explicit methods of the pressio ode aim to solve systems of the form\+: \[ \frac{d \boldsymbol{y}}{dt} = \boldsymbol{f}(\boldsymbol{y},t; ...) \]
