Provides metaprogramming functionalities that are useful for a variety of purposes and are one of the fundamental building blocks for the other parts of the pressio library. 

If you are familiar with the {\ttfamily $<$type\+\_\+traits$>$} header from the standard library, the {\ttfamily pressio/mpl} will look familiar too. Some parts of {\ttfamily pressio/mpl} have been adapted from the \href{http://sbabbi.github.io/tinympl}{\texttt{ tinympl project}}. The tinympl project appears to be no longer maintained.\hypertarget{md_pages_components_mpl_autotoc_md10}{}\doxysection{Content}\label{md_pages_components_mpl_autotoc_md10}
The following is a {\itshape partial} list only intended to provide a general idea of the supported features.
