pressio C++ library¶

Advancing reduced order models (ROMs) for dynamical systems in science and engineering

This is the documentation of the C++ library, one component of the Pressio ecosystem.

Description

Header(s)

mpl

metaprogramming functionalities

<pressio/mpl.hpp>

utils

logger, constants, etc

<pressio/utils.hpp>

type_traits

type traits and detection

<pressio/type_traits.hpp>

expressions

Expr templates for span, diagonal, subspan, etc.

<pressio/expressions.hpp>

ops

specializations of shared-memory and distributed linear algebra kernels

<pressio/ops.hpp>

qr

QR factorization functionalities

<pressio/qr.hpp>

solvers_linear

linear dense solvers

<pressio/solvers_linear.hpp>

solvers_nonlinear

Including Newton-Raphson, Gauss-Newton, Levenberg-Marquardt

<pressio/solvers_nonlinear.hpp>

ode

explicit time steppers
implicit time steppers
advancers

<pressio/ode_steppers_explicit.hpp>
<pressio/ode_steppers_implicit.hpp>
<pressio/ode_advancers.hpp>
<pressio/ode.hpp> to include all

rom

decoder
Galerkin
LSPG: steady/unsteady
WLS

<pressio/rom_decoder.hpp>
<pressio/rom_galerkin.hpp>
<pressio/rom_lspg.hpp>
<pressio/rom_wls.hpp>
<pressio/rom.hpp> to include all
